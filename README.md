#  Toko Baju – Website E-Commerce Sederhana  
Proyek UAS Mata Kuliah Pemrograman Web  
NIM - 159 - 149 - 170
Kelompok 5 — D4 Manajemen Informatika

Website ini adalah aplikasi toko baju sederhana berbasis web yang dibangun menggunakan **HTML, CSS, JavaScript, PHP, dan MySQL.  
Fitur mencakup katalog produk, keranjang belanja, checkout, manajemen produk untuk admin, dan sistem login sederhana.

---

##  Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP Native  
- **Database:** MySQL  
- **Server:** Localhost (XAMPP)  
- **Version Control:** Git & GitHub  

---

##  Fitur Utama

### Untuk Pengguna (User)
- Melihat daftar produk  
- Melihat detail produk  
- Menambah produk ke keranjang  
- Mengedit jumlah dan menghapus item dari keranjang  
- Checkout dan pengisian data pembeli  
- Riwayat pesanan (opsional)  

---

### Untuk Admin
Admin dapat mengelola seluruh konten website melalui halaman dashboard:

- Login Admin  
- Dashboard Admin  
- CRUD Produk
  - Tambah produk  
  - Edit produk  
  - Hapus produk  
- Upload gambar produk  
- Proteksi halaman admin (session login)  
- Logout Admin
